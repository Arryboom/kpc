基础用法

Steps嵌套Step即可实现步骤条，用v-model双向绑定当前指向哪一步，索引从0开始

index.vdt index.styl index.js

属性

Steps

属性 说明 类型 默认值
value 标识当前第几步，从0开始 Number undefined
status 步骤条的状态 "normal" | "error" "normal"
type 步骤条样式 "default" | "line" "default"
