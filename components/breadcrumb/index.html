基础用法

指定分隔符

通过separator属性可以指定分隔符，该属性值会使用innerHTML写入，所以可以使用任意html字符串。

使用innerHTML存在XSS风险，需要自行规避

index.vdt index.styl index.js

属性

Breadcrumb

属性 说明 类型 默认值
separator 指定分隔符 String undefined

BreadcrumbItem

属性 说明 类型 默认值
to 指定超链接地址 String undefined
separator 指定分隔符，会覆盖Breadcrumb中的定义 String ">"
